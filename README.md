# PVモジュール温度特性計算ツール (機械的完工チェック)

## 概要
このWebアプリケーションは、太陽光パネル（PVモジュール）の温度変化に伴う開放電圧（Voc）、動作電圧（Vmp）、最大出力（Pmax）の変動を計算するツールです。
ユーザーが提供した「機械的完工チェック（電気的確認）」のPDF仕様に基づき、入力値（オレンジ色のセクション）から計算結果（青色のセクション）を動的に生成します。

## 機能

1.  **温度特性計算**
    *   任意のモジュール温度における電圧・出力を算出
    *   直列数（ストリング構成）に基づいたシステム電圧の計算
    *   システム最大電圧（デフォルト1500V）超過時の警告表示

2.  **計算モード切替**
    *   **標準モード (デフォルト)**: 電圧補正に温度係数 $\beta$、出力補正に $\gamma$ を使用（IEC標準/業界推奨）。
    *   **互換モード**: 電圧補正・出力補正ともに $\gamma$ を使用（提供されたPDF/Excelシートの計算結果を再現するため）。

3.  **UI/UX**
    *   レスポンシブデザイン
    *   行の追加・削除機能（任意の温度ポイントを追加可能）
    *   計算結果の視覚的フィードバック（アニメーション）
    *   入力パラメータの永続化（リロード時はデフォルト値に戻りますが、コード内でデフォルト値を設定可能）

## 計算ロジック

$\Delta T = T_{ambient} - 25^\circ C$

### 標準モード
*   $Voc(T) = Voc_{stc} \times (1 + \beta \times \Delta T)$
*   $Vmp(T) = Vmp_{stc} \times (1 + \beta \times \Delta T)$
*   $Pmax(T) = Pmax_{stc} \times (1 + \gamma \times \Delta T)$

### 互換モード
*   $Voc(T) = Voc_{stc} \times (1 + \gamma \times \Delta T)$
*   $Vmp(T) = Vmp_{stc} \times (1 + \gamma \times \Delta T)$
*   $Pmax(T) = Pmax_{stc} \times (1 + \gamma \times \Delta T)$

※ 係数は入力値（%）を100で除算して使用します。
